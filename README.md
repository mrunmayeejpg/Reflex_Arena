**Reflex Arena** is a fun mobile game built with **React Native** and **Expo**.  
It includes mini-challenges like:

- **Red Dot Challenge**: Tap disappearing red dots as fast as you can.  
- **Whack-A-Mole**: Tap moles before they disappear!
- **Reaction Test**: Tap when the screen turns green!  

The game progressively increases in difficulty, testing your reflexes and reaction speed.

## Features

- Progressive difficulty for mini-games
- Smooth animations with **react-native-reanimated**
- Haptic feedback on interactions
- Mobile-friendly UI with **expo-linear-gradient**

## Installation

1. Clone the repo:

git clone https://github.com/USERNAME/Reflex_Arena.git
cd Reflex_Arena


Install dependencies:

npm install
or
yarn install

# Run in development:

npx expo start

# Build APK for Android:

eas build --platform android --profile production
# Tech Stack
- React Native & Expo
- TypeScript
- React Query
- Reanimated & Expo Haptics
